# Source documents

Drop real clinical source PDFs here for the authoring agent to ingest
(e.g. the 2023 International Evidence-based PCOS Guideline, a research paper,
or your organization's existing protocol doc).

The authoring agent also accepts PDFs uploaded at runtime in the browser — this
folder is for committed demo samples so the hosted app has something to read
out of the box.

Opus 4.8 reads these natively via API document content blocks (base64); nothing
is pre-parsed or chunked.
